# Data Cleaning & Preprocessing using Python

## Objective
This project demonstrates data cleaning and preprocessing techniques using Python and pandas.
It is part of an internship assignment (Task 1).

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Steps Performed
- Data ingestion and inspection
- Duplicate removal
- Column management
- Missing value handling
- Data type correction
- Format standardization

## Output
A clean, analysis-ready dataset (`cleaned_sales_data.csv`).
